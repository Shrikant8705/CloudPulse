# CloudPulse - Cloudburst Prediction System

Machine learning system for predicting cloudbursts and extreme weather events.

## Features

- ML-based weather prediction
- Real-time atmospheric analysis
- Automated alerts
- Data pipeline for processing
- Python backend

## Tech Stack

- Python
- TensorFlow & Scikit-learn
- Flask/FastAPI
- PostgreSQL

## Quick Start

1. Clone the repo:
```bash
git clone https://github.com/Shrikant8705/CloudPulse.git
cd CloudPulse
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Download data:
```bash
python scripts/download_data.py
```

4. Run predictions:
```bash
python backend/predict.py
```

## Use Cases

- Disaster preparedness
- Early weather warnings
- Risk assessment
- Climate research
